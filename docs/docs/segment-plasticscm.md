## What

Plastic SCM segment

## Sample Configuration

```json
{
  "type": "plasticscm",
  "style": "powerline",
  "powerline_symbol": "\uE0B0",
  "foreground": "#193549",
  "background": "#ffeb3b",
  "properties": {
    "cmPath": "cm",
    "scmFolder": ".plastic",
  }
}
```

## Properties

- binary: `string` - the path of the binary, defaults to `cm`
